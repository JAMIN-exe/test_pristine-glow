# Pristine Glow

A responsive skincare landing page built to practice and showcase front-end development skills using HTML and CSS.

## Overview

Pristine Glow is a fictional skincare brand website designed with a clean and modern aesthetic. The project focuses on responsive layouts, semantic HTML, and effective CSS styling.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Semantic HTML5 structure
* Modern CSS using Flexbox and Grid
* Product showcase section
* About section
* Contact form UI
* Clean and accessible layout

## Built With

* HTML5
* CSS3

## Sections

* Hero
* About
* Products
* Why Choose Us
* Contact
* Footer

## Learning Goals

This project was created to practice:

* Responsive web design
* CSS Grid and Flexbox
* Semantic HTML structure
* Layout design
* UI styling and spacing
* Mobile responsiveness
